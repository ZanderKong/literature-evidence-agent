# Roadmap — 文献证据 Agent

## Round 1（当前）

本地 PDF 导入 → 解析 → 主张提取 → 人工复核 → SQLite 证据库 → 检索导出

**明确不做**（记录以备后续轮次）：

- 在线论文检索（CrossRef, OpenAlex）
- 专利检索（EPO）
- 专利族识别
- 网页抓取
- OCR 批处理
- 扫描版 PDF 的完整支持
- 图片视觉理解
- 曲线数字化
- 表格数值全量提取
- 外部实验数值数据库
- 内部实验数据管理
- 跨论文定量合并
- 向量数据库
- 图数据库
- 自动研究结论
- 自动实验方案
- 多 Agent 协作
- 生产级 Web 前端
- 用户账户和权限系统

## Round 2（候选）

- 在线论文检索（CrossRef / OpenAlex）
- 专利检索（EPO API）
- 多文献交叉引用追踪
- 改进的 PDF 表格提取

## Round 3（候选）

- 曲线数字化
- 向量数据库（语义检索）
- 跨论文定量数据合并
- 图数据库（知识图谱）
